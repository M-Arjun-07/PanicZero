  Written: 2026-04-27

  This file lists every pip package that must be installed, every Python
  stdlib module that is used (no install needed), and every internal
  project module, grouped by sub-project.

  HOW TO INSTALL EVERYTHING IN ONE SHOT
  ──────────────────────────────────────
  Run these three commands from the project ROOT folder:

    pip install -r backend/requirements.txt
    pip install -r yolo-model/requirements.txt
    pip install -r audio-model/requirements.txt

  Or install all at once (creates duplicates but is harmless):

    pip install fastapi uvicorn pydantic websockets fakeredis firebase-admin httpx ^
                ultralytics opencv-python requests ^
                tensorflow sounddevice numpy scipy openai-whisper

  Python version requirement: 3.10 or higher (uses match/case syntax style
  and modern typing hints like X | Y).

  SECTION 3 — AUDIO MODEL  (audio-model/)
================================================================================

  Entry point : audio-model/audio_detector.py
  Run command : python audio_detector.py --source mic
                python audio_detector.py --source path/to/file.wav

  ── 3A. THIRD-PARTY PACKAGES (pip install required) ─────────────────────────

  Package            pip install name        Used in file(s)          Why needed
  ─────────────────  ──────────────────────  ───────────────────────  ─────────────────────────────────────
  tensorflow         tensorflow>=2.10.0      audio_detector.py        Run YAMNet TFLite model inference
  sounddevice        sounddevice>=0.4.6      audio_detector.py        Capture live microphone audio stream
  numpy              numpy>=1.23.0           audio_detector.py        Waveform array manipulation
  scipy              scipy>=1.9.0            audio_detector.py        Read .wav files (scipy.io.wavfile)
  requests           requests>=2.28.0        audio_detector.py        POST alerts to FastAPI backend
  openai-whisper     openai-whisper          audio_detector.py        Speech-to-text for phrase detection
                     (NOT whisper-jax —                               NOTE: install with:
                      use openai-whisper)                               pip install openai-whisper

  Install command:
    pip install tensorflow sounddevice "numpy>=1.23.0" scipy requests openai-whisper

  NOTE on Whisper: The requirements.txt says "whisper-jax" but the code does
  "import whisper" which maps to the openai-whisper package. Use:
    pip install openai-whisper
  (not whisper-jax, which has a different API)

  Model files required (auto-downloaded on first run if missing):
    audio-model/yamnet.tflite         — YAMNet TFLite audio classification model
    audio-model/yamnet_class_map.csv  — 521 class labels for YAMNet output
    Whisper "tiny" model              — Downloaded automatically by openai-whisper
                                        on first call to whisper.load_model("tiny")

  urllib.request     Download yamnet.tflite + class map if missing

  ── 4A. PYTHON STANDARD LIBRARY ONLY ─────────────────────────────────────────

================================================================================
  SECTION 5 — QUICK INSTALL CHEATSHEET
================================================================================

  Run from the PROJECT ROOT in this exact order:

  STEP 1 — Backend
  ─────────────────
    cd backend
    pip install fastapi uvicorn pydantic websockets httpx firebase-admin "fakeredis[aioredis]"

  STEP 2 — YOLO Camera Model
  ──────────────────────────
    cd ../yolo-model
    pip install ultralytics opencv-python numpy requests

  STEP 3 — Audio Model
  ──────────────────────
    cd ../audio-model
    pip install tensorflow sounddevice "numpy>=1.23.0" scipy requests openai-whisper

  STEP 4 — Verify backend starts
  ────────────────────────────────
    cd ../backend
    python main.py
    → Open http://127.0.0.1:8000/docs to confirm all endpoints appear

  STEP 5 — Place your Firebase credentials file
  ──────────────────────────────────────────────
    Copy firebase-credentials.json into backend/
    (If it's already there, you're good.)

  ── VIRTUAL ENVIRONMENT (recommended) ────────────────────────────────────────

    python -m venv venv
    venv\Scripts\activate          # Windows
    source venv/bin/activate       # macOS / Linux
    pip install <packages above>


================================================================================
  SECTION 6 — COMMON ERRORS & FIXES
================================================================================

  ERROR: "No module named 'whisper'"
  FIX:   pip install openai-whisper   (NOT whisper, NOT whisper-jax)

  ERROR: "No module named 'cv2'"
  FIX:   pip install opencv-python

  ERROR: "No module named 'ultralytics'"
  FIX:   pip install ultralytics

  ERROR: "No module named 'firebase_admin'"
  FIX:   pip install firebase-admin
         AND ensure backend/firebase-credentials.json exists

  ERROR: "No module named 'fakeredis'"
  FIX:   pip install "fakeredis[aioredis]"   (include the [aioredis] extra)

  ERROR: "No module named 'sounddevice'"
  FIX:   pip install sounddevice
         On Windows you may also need: pip install sounddevice pyaudio

  ERROR: "ModuleNotFoundError: No module named 'tensorflow'"
  FIX:   pip install tensorflow
         Note: tensorflow requires Python 3.8–3.11. Does NOT support 3.12+ yet.
         If on Python 3.12, use: pip install tensorflow-cpu

  ERROR: "DefaultCredentialsError" from Firebase
  FIX:   Make sure backend/firebase-credentials.json is present and valid.
         Download it from Firebase Console → Project Settings → Service Accounts.
